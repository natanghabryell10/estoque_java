# estoque_java


 Sistema de Estoque em Java

Projeto simples de controle de estoque feito em Java no terminal.
CRUD (Create, Read, Update, Delete)

Funcionalidades
- Adicionar produtos
- Listar produtos
- Atualizar quantidade
- Remover produtos

Como executar
1. Compile:
javac Estoque.java

2. Rode:
java Estoque
